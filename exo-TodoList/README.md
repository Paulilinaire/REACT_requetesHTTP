# REACT_requetesHTTP

Il faut installer server j-son de façon globale : ``` npm install -g json-server```

Après création d'un fichier json, on  démarre le serveur web avec ```json-server --watch bdd.json --port 5000```

- GET /personnes
- POST /personnes
- PUT /personnes/:id
- DELETE /personnes/:id
